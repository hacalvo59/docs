# 📘 **86\_normas\_seguridad\_fresas\_plaquitas.md**

**Normas oficiales de seguridad para fresas de plaquitas — CNC‑CAT / LINCAT**

## ⭐ 1. Propósito

Definir las **normas de seguridad para fresas de plaquitas** utilizadas en máquinas CNC‑CAT, garantizando:

- seguridad del operador

- integridad del husillo

- estabilidad del mecanizado

- fijación correcta de plaquitas

- precisión en desbaste y acabado

- cumplimiento industrial

Las fresas de plaquitas son potentes, versátiles y peligrosas si se usan mal. Si una plaquita sale despedida… *no hay segunda oportunidad*. Estas normas son **críticas** y deben cumplirse SIEMPRE.

## ⭐ 2. Principios fundamentales

- **La fijación es vida** — una plaquita floja es un proyectil.

- **La geometría debe ser correcta** — desgaste = vibración.

- **La refrigeración es obligatoria** — calor = rotura.

- **La rigidez es esencial** — fuerzas laterales enormes.

## ⭐ 3. Tipos de fresas de plaquitas cubiertas

- **Fresas de plaquitas para aluminio**

- **Fresas de plaquitas para acero**

- **Fresas de plaquitas de alto avance**

- **Fresas de plaquitas de acabado**

## ⭐ 4. Requisitos previos

Antes de montar una fresa de plaquitas:

- **Revisar torque de tornillos**

- **Revisar desgaste de plaquitas**

- **Revisar asiento limpio**

- **Revisar longitud expuesta**

- **Revisar balanceo**

## ⭐ 5. Seguridad en fresas de plaquitas para aluminio

- **Prohibido usar sin evacuación de viruta**

- **Prohibido usar sin lubricación mínima**

- **Prohibido usar plaquitas con filo pulido dañado**

- Validación obligatoria:

  - filo

  - vibración

  - temperatura

## ⭐ 6. Seguridad en fresas de plaquitas para acero

- **Prohibido usar sin refrigeración abundante**

- **Prohibido usar plaquitas con desgaste térmico**

- **Prohibido usar sin rigidez suficiente**

- Validación obligatoria:

  - ruido

  - vibración

  - estabilidad

## ⭐ 7. Seguridad en fresas de plaquitas de alto avance

- **Prohibido usar sin control de vibración**

- **Prohibido usar sin rigidez estructural**

- **Prohibido usar sin refrigeración dirigida**

- Validación obligatoria:

  - estabilidad

  - vibración

  - temperatura

## ⭐ 8. Seguridad en operación

- **Prohibido usar RPM incorrectas**

- **Prohibido usar avances incorrectos**

- **Prohibido usar plaquitas desbalanceadas**

- **Prohibido usar herramientas demasiado largas**

- Validación obligatoria:

  - ruido

  - vibración

  - temperatura

## ⭐ 9. Prohibiciones absolutas

- usar plaquitas sucias

- usar plaquitas desgastadas

- usar plaquitas flojas

- usar plaquitas sin torque

- usar fresas sin refrigeración

- usar fresas sin evacuación

- usar RPM incorrectas

- usar avances incorrectos

- ignorar vibración

- ignorar ruido

- ignorar temperatura

- operar sin validar

- operar sin registrar

## ⭐ 10. Objetivo final

Estas normas garantizan que las fresas de plaquitas CNC‑CAT sean:

- seguras

- estables

- precisas

- profesionales

- industriales

- trazables

- confiables

Son las normas oficiales de seguridad de fresas de plaquitas del ecosistema CNC‑CAT.
