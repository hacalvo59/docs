# 📘 **105\_normas\_seguridad\_fresas\_interdimensionales.md**

**Normas oficiales de seguridad para fresas interdimensionales — CNC‑CAT / LINCAT**

## ⭐ 1. Propósito

Definir las normas de seguridad para fresas que:

- existen en múltiples universos,

- cambian de forma según el observador,

- pueden aparecer en tu bolsillo sin aviso,

- y que Hugo jamás debería acercar al husillo chico.

## ⭐ 2. Principios fundamentales

- **La fresa interdimensional cambia de tamaño sola**

- **La fresa interdimensional puede abrir portales**

- **La fresa interdimensional NO debe tocar el spindle**

- **La fresa interdimensional puede duplicarse**

## ⭐ 3. Reglas universales de operación

- **Prohibido usarla sin casco cuántico**

- **Prohibido dejarla sola en el taller**

- **Prohibido usarla para cortar fruta**

- **Prohibido usarla para impresionar a Norma**

- **Prohibido usarla para viajar en el tiempo**

## ⭐ 4. Validaciones previas

Antes de usar una fresa interdimensional:

- **Revisar que esté en esta dimensión**

- **Revisar que no esté en otra**

- **Revisar que no esté duplicada**

- **Revisar que Norma no esté mirando**

## ⭐ 5. Prohibiciones absolutas

- usar fresas interdimensionales para mecanizar

- usar fresas interdimensionales como palpador

- usar fresas interdimensionales como refrigerante

- usar fresas interdimensionales como fresa de fruta

- usar fresas interdimensionales como fresa cuántica

- usar fresas interdimensionales como fresa real

- usar fresas interdimensionales sin permiso del multiverso

## ⭐ 6. Objetivo final

Garantizar que **todas las fresas interdimensionales**:

- estén documentadas

- estén protegidas

- estén aprobadas por los dioses del CNC

- y sobre todo… **que Hugo no las meta en el husillo por error**
