# 📘 **107\_normas\_seguridad\_fresas\_malditas.md**

**Normas oficiales de seguridad para fresas malditas — CNC‑CAT / LINCAT**

## ⭐ 1. Propósito

Definir las normas de seguridad para fresas que:

- traen mala suerte,

- rompen el husillo con solo mirarlo,

- vibran aunque estén balanceadas,

- y hacen que Norma te mire con cara de “¿qué hiciste ahora?”.

## ⭐ 2. Principios fundamentales

- **La fresa maldita siempre vibra**

- **La fresa maldita siempre rompe algo**

- **La fresa maldita siempre aparece donde no debe**

- **La fresa maldita NO debe tocar el husillo chico**

## ⭐ 3. Reglas universales de operación

- **Prohibido usarla sin un exorcista industrial**

- **Prohibido dejarla cerca de otras fresas**

- **Prohibido usarla para cortar fruta**

- **Prohibido usarla para impresionar a Norma**

## ⭐ 4. Validaciones previas

Antes de usar una fresa maldita:

- **Revisar que no esté poseída**

- **Revisar que no esté oxidada**

- **Revisar que no esté riéndose sola**

- **Revisar que Norma no esté mirando**

## ⭐ 5. Prohibiciones absolutas

- usar fresas malditas para mecanizar

- usar fresas malditas como palpador

- usar fresas malditas como refrigerante

- usar fresas malditas como fresa de fruta

- usar fresas malditas como fresa cuántica

- usar fresas malditas como fresa interdimensional

- usar fresas malditas sin permiso del sacerdote CNC

## ⭐ 6. Objetivo final

Garantizar que **todas las fresas malditas**:

- estén documentadas

- estén encerradas

- estén lejos del husillo

- estén lejos de Hugo

- y sobre todo… **que no vuelvan a aparecer en el taller nunca más**
